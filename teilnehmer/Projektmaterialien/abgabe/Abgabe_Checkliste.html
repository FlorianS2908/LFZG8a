Abgabe-Checkliste HTML-und-CSS

Diese Liste kann vor der Abgabe einmal Punkt für Punkt durchgegangen werden. Wenn ein Punkt nicht erfüllt ist, sollte die Ursache kurz notiert und korrigiert werden.

Technik

  • Keine aktiven .CSS-Dateien
  • Keine CSS-Imports
  • Externe CSS-Datei vorhanden
  • Relative Pfade geprüft

Layout

  • Flexbox sinnvoll verwendet
  • Grid sinnvoll verwendet
  • Mobile-First-Basis erkennbar
  • Bilder laufen nicht aus dem Container

Qualität

  • CSS-Datei lesbar gegliedert
  • Kontraste geprüft
  • DevTools-Test durchgeführt
  • Kurzbeschreibung vorhanden

Abgabe

  • Startdatei heißt index.html
  • ZIP enthält alle benötigten Dateien
  • Unnötige Dateien entfernt
  • Projekt lässt sich lokal öffnen
