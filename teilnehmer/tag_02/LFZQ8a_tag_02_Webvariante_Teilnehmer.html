Typisches Flexbox-Muster

Ein sauberer Flex-Container braucht nicht viele Eigenschaften. Wichtig ist, dass die Wirkung jeder Eigenschaft verstanden wird.

.card-row {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}

.card {
  flex: 1 1 260px;
}

Einordnung

Wann Flexbox?

Flexbox wird genutzt, wenn Elemente in einer Richtung verteilt werden sollen. Für vollständige Seitenraster ist CSS Grid meist besser geeignet.

Flexbox

  • eine Achse
  • Navigation
  • Kartenreihe
  • Ausrichtung

Grid

  • zwei Achsen
  • Seitenlayout
  • Dashboard
  • Rasterstruktur
