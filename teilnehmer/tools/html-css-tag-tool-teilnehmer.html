HTML-Tags verstehen, Syntax ansehen und passende CSS-Styles direkt vergleichen.

Die Tags sind nach Themenbereichen gruppiert. Jedes Tag kann aufgeklappt werden. In der Detailansicht wird das Tag erklärt, als HTML-Code gezeigt und mit passenden CSS-Varianten in Aktion dargestellt.

So funktioniert die Übersicht: Öffne eine Tag-Karte, wähle eine CSS-Variante über die Tabs und vergleiche Vorschau und Code. Die Umschaltung erfolgt nur mit HTML und CSS.
Tag 1

Tag 1: HTML-Grundstruktur und CSS-Basics

Der erste Tag legt das Fundament: HTML-Dokumente werden sauber aufgebaut, Inhalte semantisch gegliedert und mit grundlegenden CSS-Eigenschaften gestaltet.

<html>Dokument

<html>

Bedeutung

Das html-Element ist das Wurzelelement einer HTML-Seite.

Typische Verwendung

Es umschließt head und body und legt über lang die Sprache fest.

HTML-Syntax

<html lang="de">
  <head>...</head>
  <body>...</body>
</html>

Passende CSS-Selektoren

html
html[lang="de"]

CSS-Varianten in Aktion
