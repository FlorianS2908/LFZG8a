# Archiv

Dieser Ordner enthaelt alte Root-Materialien und Doppelstrukturen, die nicht mehr direkt fuer den regulaeren Start des Tools benoetigt werden.

Aktive Kursmaterialien liegen in:

- `dozent/`
- `teilnehmer/`

Die Electron-App bleibt separat in:

- `desktop-app/`

Der Ordner `_archiv/root_material_alt/` dient nur als Ruecklage, falls alte Varianten, Zwischenstaende oder Pruefdateien spaeter noch verglichen werden muessen.
