Umstellung

Moderne CSS-Grundlagen: fachliches Mapping

Die Umstellung bedeutet nicht, dass die früheren CSS-Themen ersatzlos gestrichen werden. Sie werden nur mit aktuellen CSS-Mitteln behandelt, damit die Teilnehmenden keine zusätzliche Kompilierung benötigen.

Alter SchwerpunktNeuer CSS-SchwerpunktUnterrichtliche Umsetzung
CSS Custom PropertiesCSS Custom PropertiesFarben, Abstände, Radius und Schatten zentral im :root definieren.
MixinsKomponentenklassen und kleine UtilitiesWiederkehrende Muster wie Button, Card oder Container als Klasse anlegen.
Verschachtelte Selektorenklare KomponentenstrukturSelektoren kurz halten und Klassen bewusst benennen.
Operationencalc(), min(), max(), clamp()Breiten, Schriftgrößen und Abstände flexibel berechnen lassen.
Layout-HelferFlexbox und GridFlexbox für Navigationen und Ausrichtungen, Grid für Seiten- und Dashboardlayouts.
Kompilierungdirekt nutzbares CSSWeniger Toolchain. Der Browser interpretiert die CSS-Datei direkt.

Beispiel

CSS Custom Properties

:root {
  --color-primary: #005b96;
  --space-md: 1rem;
}

.button {
  background: var(--color-primary);
  padding: var(--space-md);
}

Fluid CSS

h1 {
  font-size: clamp(2rem, 5vw, 4rem);
}

.container {
  width: min(100% - 2rem, 1120px);
}
