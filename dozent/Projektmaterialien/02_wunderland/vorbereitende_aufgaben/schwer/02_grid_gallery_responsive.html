Responsive Galerie mit Grid bauen

Hero-Bereich, Galerie, Bildkarten, Buttons, Farben, Abstaende und responsive Kartenraster.

Arbeitsauftrag

Gestalte die Galerie mit CSS Grid, minmax und gap. Die Karten sollen bei schmalen Viewports automatisch umbrechen.

Projektbezug

Arbeite im Projekt Wunderland CSS Only und pruefe dein Ergebnis direkt im Browser.

Projektmaterial oeffnen

Loesungshinweise

Erwartet wird ein robustes Raster mit repeat(auto-fit oder auto-fill, minmax(...)).
