Code lesen

Grundstruktur einer CSS-Datei

Zuerst werden globale Werte festgelegt. Danach folgen allgemeine Elemente und konkrete Komponenten.

:root {
  --farbe-akzent: #00abc7;
  --abstand: 1rem;
}

.card {
  padding: var(--abstand);
  border-radius: 1rem;
  border: 1px solid #dce9ec;
}
