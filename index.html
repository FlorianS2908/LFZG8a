LF ZQ8a: Modernes CSS statt LESS

Das Lernfeld wurde inhaltlich neu aufgebaut. Der bisherige LESS-Schwerpunkt wird durch modernes CSS ersetzt. Dabei werden Responsive Design, Flexbox, Grid, Custom Properties, moderne Einheiten und eine lokale Webfont-Strategie zusammengeführt.

Die Unterlagen sind bewusst praxisnah formuliert. Es geht nicht darum, CSS auswendig zu lernen, sondern die Wirkung im Browser nachvollziehen zu können. Deshalb werden die einzelnen Themen immer zuerst kurz eingeordnet und anschließend direkt am Projekt ausprobiert.

Flexbox, Navigation und Komponenten

Tag 3

CSS Grid und Seitenlayout

Nach Flexbox wird das Layout größer gedacht. Mit Grid lassen sich Bereiche wie Header, Sidebar, Content und Aside klar aufteilen, ohne dass dafür Tabellen oder unnötige Wrapper verwendet werden müssen.

Tag 4

Responsive Design und Media Queries

Die Seite soll nicht nur auf dem eigenen Bildschirm funktionieren. Deshalb wird an diesem Tag gezielt mit verschiedenen Viewports gearbeitet. Ausgangspunkt ist immer die kleinste sinnvolle Ansicht.
